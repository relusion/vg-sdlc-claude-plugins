# Feature-Verify — Artifact Template (`verification-report.md`)

Write-time template for the `verify` skill (orchestrator: `SKILL.md`). Load it at Stage 3.3; do not reconstruct the format from memory. The report is **cumulative**: each milestone run updates its journey's section; the pre-handover pass completes the whole-system sections. The Per-Feature Status rules (derived roll-up, ownership, milestone `partial`) are normative in `SKILL.md` Stage 3.3 — the caption below is the short reminder.

---
