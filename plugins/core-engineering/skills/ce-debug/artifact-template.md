# Feature-Debug — Artifact Template (`diagnosis.md`)

Write-time template for the `debug` skill (orchestrator: `SKILL.md`). Load it at Stage 4; do not reconstruct the format from memory. `diagnosis.md` is **cumulative** across the plan's features — each run appends a `DX-N` entry; prior ones stay. (Under auto-build, write the per-feature `specs/<id>/diagnosis.md` instead.)
