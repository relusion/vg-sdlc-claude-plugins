# Architecture Golden Fixture

A compact, written multi-feature plan for a team invitation workflow with
role-based authorization. It gives `/core-engineering:ce-architecture` a stable
plan, accepted ADR, runtime evidence, trust boundaries, an interaction contract,
and literal quality targets. The `architecture/` directory is the separately
frozen schema-v2 golden output used by the deterministic architecture-lint gate;
the runnable input fixture under `evals/fixtures/architecture-ready-plan/`
remains unseeded so EVAL-020 can stop at Scope Confirmation.
