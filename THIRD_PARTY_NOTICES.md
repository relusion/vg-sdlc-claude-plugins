# Third-Party Notices

The generated package-name corpus distributed with the `core-engineering`
plugin incorporates third-party data. Its attribution, license text,
modification statement, and source details are in the notice that ships with
the plugin:

- [`plugins/core-engineering/THIRD_PARTY_NOTICES.md`](./plugins/core-engineering/THIRD_PARTY_NOTICES.md)

The corpus contains package identifiers for offline typo-squatting checks. It
does not bundle the source code of the named packages.
